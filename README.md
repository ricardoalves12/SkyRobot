# This project will show you a step by step on how to build a friendly robot that detects infrared movements then waves at you.

Material used in this project:
Cardboard box
Microcontroller (Arduino Uno)
LCD screen (16x2) 
Servo motor
Infrared sensor (IR) 
googly eyes
